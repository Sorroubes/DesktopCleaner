# Desktop Cleaner

Легка утиліта для автоматичного очищення робочого столу від старих файлів. Додає пункт у контекстне меню робочого столу, щоб перемістити файли старше заданого строку в окрему папку одним кліком.

## 🎯 Мета та призначення
Програма призначена для користувачів Windows, які хочуть підтримувати робочий стіл у чистоті без зайвих зусиль. Вона переміщує файли (старші за N днів) у папку архіву, пропускаючи ярлики (.lnk, .url) та папки. Ідеально для тих, хто накопичує файли на столі, але не хоче вручну сортувати.

**Ключові можливості:**
- Налаштування через просте GUI: кількість днів та назва архівної папки.
- Автоматичне перейменування файлів при конфліктах імен.
- Робота **тільки за командою** — без фонових процесів, щоб не красти ресурси CPU/пам'яті системи.

## 🚀 Встановлення та використання
**Готовий інсталятор!** У репозиторії є `DesktopCleaner.exe` — просто запустіть для встановлення та тестування.

Альтернатива (для розробників):
1. Завантажте файли: `cleaner.py`, `settings.py`, `config.json`.
2. Запустіть `settings.py` для налаштувань.
3. Скомпілюйте: `pyinstaller --onefile --windowed --icon=icon.ico cleaner.py`.
4. Додайте .exe до контекстного меню робочого столу.

Після встановлення: клік правою кнопкою на столі → "Clean Desktop" → готово!

## ⚠️ Відомі недоліки

Програма **виконує свої основні функції**, але має кілька суттєвих обмежень, які я поки що не зміг повністю доопрацювати:

- Файли потім все одно потрібно сортувати. Але принаймні робочий стіл стане чище. 
- Механізм повідомлень про виконану роботу (через `plyer.notification`) **не працює коректно**: код виклику сповіщень є, але на практиці тости не з’являються.
- Програма **не інтегрується в нове контекстне меню Windows 11** – пункт меню доступний лише в класичному контекстному меню (через «Показати додаткові параметри»).

Я дуже сподіваюсь, що хтось із спільноти допоможе **допрацювати ці частини**: виправити систему сповіщень і реалізувати повноцінну інтеграцію з контекстним меню Windows 11.
